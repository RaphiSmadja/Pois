# Pois

Pois is a java project to calculate the density of a population.

## Installation

Use maven to install all dependencies

```bash
maven clean install
maven test
```

## CI

I used jenkins to create a CI and run pipeline but we must install some tools :
- maven
- jdk 8 or 11
